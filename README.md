# 🏔️ Saurav Trek & Tours

A modern, responsive travel and tour booking website for Uttarakhand tourism, Char Dham Yatra, trekking adventures, and customized travel packages.

## 🌟 Features

* Responsive modern UI
* Char Dham Yatra packages
* Kedarnath & Badrinath tours
* Trekking package showcase
* Online enquiry form
* Booking modal popup
* WhatsApp integration
* Google Sheets lead storage
* Email notifications for bookings
* Mobile-friendly design
* Fast loading and SEO optimized

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5
* Font Awesome
* AOS Animations
* Google Apps Script
* Google Sheets API

## 🚀 Installation

1. Clone the repository

git clone https://github.com/ItsDevArya/saurav-trek-tours.git

2. Open project folder

cd saurav-trek-tours

3. Open index.html in browser

or use VS Code Live Server.
